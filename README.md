# Overview

As a software engineer, I am continually working to strengthen my understanding of object-oriented programming and the development of interactive console applications. This project allowed me to practice designing classes, organizing program logic into methods, and managing data using Java collections while creating a practical application.

This software is a Student Management System written in Java. The application allows users to add students, view student records, search for students, update student information, delete student records, calculate class averages, display the total number of students, and identify the highest-performing student in the class. The program uses a menu-driven interface that allows users to interact with the system through the command line.

The purpose of writing this software was to gain hands-on experience with Java syntax, object-oriented programming concepts, user input handling, conditional statements, loops, methods, and the ArrayList collection class. Through this project, I improved my understanding of how data can be stored, manipulated, and displayed within a larger Java application.
